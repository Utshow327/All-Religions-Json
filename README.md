
# Major Religions JSON Texts

JSON datasets of major world religions, organized for easy use in apps, research, or offline reading.

---

## Structure

```

.
├── islam/
├── christianity/
├── hinduism/
├── judaism/
└── README.md

```

Each folder contains JSON files for that religion’s texts.

---

## Usage

- Educational apps  
- Search/indexing systems  
- AI/NLP datasets  
- Offline reading

---

## Disclaimer

For educational/research purposes only. Verify translations with authoritative sources.
